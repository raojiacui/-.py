# 注释
注释就是在代码里做的笔记，就像看书的时候做的笔记一样
# 开始到这一行的结束，都属于注释的一部分，是不会被程序识别的
# 除了在开头注释，还可以在一行内注释，因为 # 到行末都是注释的内容

"""
这是多行字符串，当做多行注释
可以换行
"""

# 循环
x = 93
n = 10
nzjc = (n - 2) * 180 / n
degree = 180 - nzjc

# 下面是循环的写法
# 首先我们定义一个变量用于计算循环的次数，通常 i j k a b c
i = 0
# while 后面的语句是条件
# 当条件成立的时候，执行代码块内的语句
# 否则，结束循环
while i < n:
    forward(x)
    right(degree)
    # 注意，一定要改变 i 的值
    # 否则循环永远不会结束（无限循环）
    i = i + 1


# 函数
# 函数是用来消除大段重复代码的功能
# 主流语言都有函数这个特性
# 函数就是一段代码的集合，用于重复使用一段代码

# 下面用例子来演示函数的定义的使用方式

# 定义一个函数，用来画一个边长为 100 的正三角形

def sjjcxy():
    n = 3
    x = 100
    nzjc = (n - 2) * 180 / n
    degree = 180 - nzjc
    i = 0
    while i < n:
        forward(x)
        right(degree)
        i = i + 1

# 上面就是定义三角形的方式，下面就可以使用了
# 注意程序是按顺序执行的，所以定义要在使用之前
# 函数名 加上一对 括号 表示调用函数（调用即为使用）

sjjcxy()

# 函数和其他普通的值是一样的，可以被赋值给另外一个变量
sanjiaoxing = sjjcxy
sanjiaoxing()

# 函数可以有一个叫做参数的概念，来增加灵活性
# 函数的参数相当于是微波炉的旋钮
def sjjcxy(x):
    n = 3
    nzjc = (n - 2) * 180 / n
    degree = 180 - nzjc
    i = 0
    while i < n:
        forward(x)
        right(degree)
        i = i + 1

sjjcxy(50)
sjjcxy(100)

# 函数可以支持多个参数，不同的参数用逗号分隔
def polygon(bianchang, bianshu):
    n = bianshu
    nzjc = (n - 2) * 180 / n
    degree = 180 - nzjc
    i = 0
    while i < n:
        forward(bianchang)
        right(degree)
        i = i + 1

polygon(100, 4)
polygon(50, 5)
polygon(20, 8)

# 变量名命名规则（函数名也是变量名）
# 变量名只能包括 字母（汉字也算字母），下划线（_），美元符号（$），数字
# 变量名不能以数字开头
# 变量名是大小写敏感的（a A 是不同的变量）
# 只有在现代 Python 中，才能使用汉字作为合法的变量名
# 变量名不能是 Python 的保留字（def while 是保留字）
# 除此之外还有很多保留字，但你现在不用关心

# 函数的默认参数

        i = i + 1
