# Python 编程思维入门

通过可视化绘图学习编程，在实践中发现问题、解决问题。

## 项目结构

```
.
├── 课上内容/          # 板书和示例代码
├── 预习/             # 预习材料和练习
├── 课后习题/         # 各课作业
├── import turtle.py  # Turtle 绘图练习
├── py               # 基础示例
└── 预习 1请用vscode打开.py  # 预习练习
```

## 学员任务清单

### 学习流程

```
预习 → 看板书 → 做作业
  ↓      ↓       ↓
自己做  自己看  独立做
```

---

### 课前任务

| 步骤 | 做什么 |
|-----|-------|
| 1 | 打开 `预习/` 文件夹 |
| 2 | 从 `预习 1` 开始，按顺序阅读 |
| 3 | 照着例子写代码，运行看效果 |
| 4 | 尝试完成文件里的作业题 |
| 5 | 记下遇到的问题 |

**注意**：没预习就直接看板书，会比较吃力

---

### 看板书

| 步骤 | 做什么 |
|-----|-------|
| 1 | 打开 `课上内容/` 文件夹 |
| 2 | 从 `课1板书` 开始，按顺序看 |
| 3 | 对照板书敲代码 |
| 4 | 运行代码，看效果 |
| 5 | 理解为什么这样写 |

---

### 课后任务

| 步骤 | 做什么 |
|-----|-------|
| 1 | 打开 `课后习题/` 对应课次文件夹 |
| 2 | 阅读作业要求 |
| 3 | 独立编写代码 |
| 4 | 运行验证结果 |
| 5 | 按时提交 |

**规则**：
- 自己做，不问同学
- 不懂就问
- 代码要能运行，结果要正确

---

### 每课 Checklist

- [ ] 预习文件看完并运行
- [ ] 预习作业尝试做
- [ ] 看板书并敲代码
- [ ] 课后作业独立完成
- [ ] 作业运行结果正确

## 运行方式

确保已安装 Python，然后在命令行运行：

```bash
# 运行课上示例
python "课上内容/基础4-课2板书.py"

# 运行预习练习
python "预习/预习 1请用vscode打开.py"

# 运行 Turtle 绘图
python "import turtle.py"
```
