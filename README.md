# Python 编程思维入门

通过可视化绘图学习编程，在实践中发现问题、解决问题。

## 环境准备

### VS Code 安装与配置

VS Code 是推荐的代码编辑器，轻量、免费、功能强大。

**VS Code 教程**：https://www.runoob.com/w3cnote/vscode-tutorial.html

#### 必装插件

| 插件名称 | 作用 |
|---------|------|
| Chinese (Simplified) | 中文界面 |
| Python | 代码提示、运行、调试 |

#### 安装步骤

1. 打开 VS Code，点击左侧扩展图标（或按 `Ctrl+Shift+X`）
2. 搜索 `Chinese`，安装「Chinese (Simplified) Language Pack」
3. 搜索 `Python`，安装微软官方的 Python 插件
4. 重启 VS Code

#### 如何运行代码

**方法一：右键运行**
- 打开 `.py` 文件
- 在代码区域右键 → 「在终端中运行 Python 文件」

**方法二：使用终端**
- 按 `` ` `` （反引号）打开终端
- 输入 `python 文件名.py` 回车

**方法三：快捷键**
- 按 `Ctrl+F5` 直接运行

## 给大一新生的重要提醒

### 关于代码

| 问题 | 说明 |
|-----|------|
| **中英文标点** | 代码中只能用英文标点 `()` `""` `,` `:`，中文标点会报错 |
| **缩进** | Python 对缩进敏感，用 Tab 或 4 个空格，不要混用 |
| **文件名** | 不要用中文，建议用英文，如 `lesson1.py` |
| **保存位置** | 把代码文件放在一个固定的文件夹里，方便查找 |

### 关于报错

| 报错类型 | 常见原因 | 解决方法 |
|---------|---------|---------|
| `SyntaxError` | 语法错误 | 检查中英文标点、括号是否匹配 |
| `NameError` | 变量名未定义 | 检查变量名是否拼写正确 |
| `IndentationError` | 缩进错误 | 检查代码对齐，统一用 Tab 或空格 |

**报不可怕**：报错是正常的，看懂报错信息就能找到问题。

### 关于学习

- **多动手**：不要只看，要亲手敲代码
- **别复制粘贴**：自己敲一遍，记忆更深刻
- **报错截图**：遇到问题记下来，方便提问
- **及时保存**：`Ctrl+S` 随时保存代码
- **备份代码**：重要的代码多备份一份

### 关于提问

遇到问题可以通过 AI 工具提问：

| 工具 | 链接 | 说明 |
|-----|------|------|
| AiStudio | https://aistudio.baidu.com/ | 百度飞桨 AI 开发平台 |
| 腾讯元宝 | https://yuanbao.tencent.com/ | 腾讯 AI 助手 |

**提问技巧**：
- 把代码和报错信息一起发给 AI
- 描述清楚你想做什么、遇到了什么问题
- AI 给出的答案要自己验证理解
